# Contributing

Thank you for contributing! Here's how:

## Setup

```bash
make install
```

## Testing

```bash
make test
```

## Code Style

```bash
make format
make lint
```

## Commits

Use conventional commits:
- `feat`: New feature
- `fix`: Bug fix
- `test`: Add tests
- `docs`: Documentation
- `chore`: Maintenance

Example: `git commit -m "feat: add CSV validation"`

## Pull Requests

1. Fork repository
2. Create feature branch
3. Commit changes
4. Push and create PR
